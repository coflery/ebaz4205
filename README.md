# 翼比特 E9 EBAZ4205

矿难后某鱼上`翼比特E9+`矿机`EBAZ4205`控制板泛滥。

该控制板使用 `Xilinx ZYNQ-7000` 系列 `XC7Z010CLG400-1` 芯片作为主控，板上引出 `JTAG` `UART` `GPIO` 可当开发板使用。

本库主要收集该控制板相关硬件资料，供二次开发参考使用。
